# 💥 Kill The Words

A fast-paced browser reflex game where you must destroy words before they flood the screen. Test your speed, focus, and accuracy — one word at a time.

![HTML](https://img.shields.io/badge/HTML-6.6%25-orange)
![CSS](https://img.shields.io/badge/CSS-46.1%25-blue)
![JavaScript](https://img.shields.io/badge/JavaScript-47.3%25-yellow)

## 🎮 About the Game

Words start appearing on screen and moving across it — your job is to eliminate them before they pile up and overflow. Each level ramps up the pace, throwing words at you faster and in trickier patterns. It's simple to understand but deeply satisfying to master.

Whether you're looking to sharpen your reflexes, improve your typing focus, or just kill some time, Kill The Words delivers an addictive challenge.

## ✨ Features

- Words appear at increasing speed with each level
- Increasingly complex patterns to keep you on your toes
- Clean, polished UI with dedicated CSS and image assets
- Modular project structure (separate `css/`, `javascript/`, and `images/` folders)
- No dependencies — runs entirely in the browser

## 🚀 Getting Started

### Prerequisites

A modern web browser is all you need.

### Run Locally

```bash
git clone https://github.com/iamparrish/Kill-The-Words.git
cd Kill-The-Words
```

Open `index.html` in your browser and start playing!

## 🗂️ Project Structure

```
Kill-The-Words/
├── index.html          # Main game page
├── css/                # Stylesheets
├── javascript/         # Game logic scripts
└── images/             # Game image assets
```
## 🕹️ How to Play

1. Launch the game by opening `index.html`.
2. Words will begin appearing on the screen.
3. Click or interact with each word to destroy it before it escapes.
4. Each level introduces more words appearing faster and in trickier spots.
5. Don't let the words flood the screen — game over if they do!

## 🛠️ Built With

- **HTML5** — Game structure
- **CSS3** — Animations, layout, and visual design
- **Vanilla JavaScript** — Word generation, collision logic, and level progression
